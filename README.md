# Kid Learn Math Using Hand Gesture Recognition

## Overview

This project uses Computer Vision to help children learn arithmetic through hand gestures.

The system detects fingers using MediaPipe and OpenCV, converts the finger counts into numbers, and performs arithmetic operations in real time.

## Features

- Real-time hand tracking
- Finger counting
- Addition
- Subtraction
- Multiplication
- Division

## Technologies

- Python
- OpenCV
- MediaPipe

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python src/main.py
```
